# 文档模板集

## 1. 变更日志模板（changelog）

文件位置：`openspec/changelog/{YYYYMMDD}-{缩写}-{描述}.md`

```markdown
# 变更日志：{简述}

**日期**：YYYY-MM-DD
**项目缩写**：global | backend | fe | sync | deploy
**类型**：功能新增 | 功能改进 | Bug 修复 | 重构 | 配置变更 | 文档更新

## 变更摘要

[1-2 句话概括这次变更做了什么]

## 变更详情

[详细描述改了什么、为什么改]

## 影响的文件/模块

| 文件/模块 | 变更类型 | 说明 |
|----------|---------|------|
| `path/to/file` | 新增/修改/删除 | 简要描述 |

## 相关设计文档

- [关联的 openspec 文档链接，如有]

## 验证方式

- [ ] 单元测试通过
- [ ] Lint 通过（如适用）
- [ ] 集成测试通过（如适用）

## 后续工作

- [ ] [如有待完成事项]
```

## 3. 架构改动文档模板（change）

文件位置：`openspec/change/{YYYYMMDD}-{缩写}-{描述}.md`

```markdown
# 架构改动：{改动名称}

**日期**：YYYY-MM-DD
**项目缩写**：global | backend | fe | sync | deploy
**状态**：草案 | 确认 | 实现中 | 已完成

## 改动原因

[为什么需要这个架构调整？当前设计有什么痛点？]

## 改动方案

### Before（现状）

[当前是怎么做的？]

### After（目标）

[改成什么样？]

## 影响范围

### 受影响的模块

| 模块 | 影响程度 | 说明 |
|------|---------|------|
| [模块名] | 重大/中等/轻微 | [如何影响] |

### 受影响的接口/协议

| 接口/协议 | 变更类型 | 说明 |
|----------|---------|------|
| [接口名] | 新增/修改/废弃 | [说明] |

## 迁移计划

[如果涉及数据迁移或 API 兼容性问题，详细说明迁移步骤]

## 风险评估

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|---------|
| [风险描述] | 高/中/低 | 高/中/低 | [怎么应对] |

## 回滚方案

[如果改动出问题，怎么回滚？]

## 验证方式

- [ ] 回归测试通过
- [ ] 性能测试通过（如适用）
- [ ] 兼容性验证通过（如适用）
```

## 4. 交接摘要模板

用于多 Agent 协作中阶段交接：

```markdown
## 交接摘要

- **阶段**：设计 / 实现 / 测试 / 审查
- **完成者**：[Agent 角色标识]
- **日期**：YYYY-MM-DD
- **完成了什么**：[1-2 句话概括]
- **改动/新建的文件**：
  - `path/to/file1` — [简要描述]
  - `path/to/file2` — [简要描述]
- **接手者注意**：
  - [需要特别注意的技术决策或坑]
- **未解决问题**：
  - [如有，列出待处理的问题]
- **下一步建议**：[建议下一步做什么]
```

## 5. AGENTS.md 骨架

当项目还没有 AGENTS.md 时，用此骨架生成：

```markdown
# {项目名称} — AI Agent 工作指南

## 项目信息

- **项目名**：{名称}
- **技术栈**：{语言、框架、数据库等}
- **运行环境**：{OS、容器、云服务等}

## 目录结构

[项目主要目录说明]

## 核心约定

1. [铁律 1]
2. [铁律 2]
3. [铁律 3]

## 各模块命令

| 模块 | 测试 | Lint | 构建 |
|------|------|------|------|
| {模块名} | `{命令}` | `{命令}` | `{命令}` |

## AI 工作流

1. 读 PROJECT.md
2. 读本文件（AGENTS.md）
3. 读最近 changelog
4. 读相关 docs/
5. 读源代码

## 关键文件位置

| 文件 | 路径 | 用途 |
|------|------|------|
| 项目入口 | `PROJECT.md` | 全项目概览 |
| Agent 规则 | `AGENTS.md` | 本文件 |
| 变更日志 | `openspec/changelog/` | 变更记录 |
| 业务规则 | `docs/` | 业务语义 |
```
