## 五、Reranker 修复说明

### v1 问题
```
RERANK_API_URL=https://dashscope.aliyuncs.com/compatible-mode/v1/reranks
→ 404 Not Found（DashScope OpenAI 兼容模式不支持 rerank）
→ 降级为 PassThroughReranker（不排序，直接截取）
→ search-1 返回了错误的检索结果（人身损害赔偿 vs 经济补偿）
```

### v2 修复
```typescript
// reranker.ts 改用 DashScope 原生 API 格式
body: JSON.stringify({
  model: 'gte-rerank',
  input: { query, documents },
  parameters: { top_n: limit, return_documents: false }
})
```
```
RERANK_API_URL=https://dashscope.aliyuncs.com/api/v1/services/rerank/text-rank/text-rank
RERANK_MODEL_ID=gte-rerank
```

### 预期效果
- search-1（经济补偿）：Reranker 能区分"经济补偿"和"人身损害赔偿"的语义差异，正确排序劳动法条款
- 所有检索查询精度提升（Dense + Sparse 结果经 Cross-Encoder 精排）
